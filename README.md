# DevOps Project 2 - Version Control with Git

## Objective
Learn the basic Git workflow and manage code using GitHub.

## Git Commands Used

### Initialize Repository
```bash
git init
```

### Check Repository Status
```bash
git status
```

### Stage Files
```bash
git add .
```

### Commit Changes
```bash
git commit -m "Initial commit"
```

### Rename Branch
```bash
git branch -M main
```

### Connect Local Repository to GitHub
```bash
git remote add origin https://github.com/USERNAME/devops-project2.git
```

### Push Code to GitHub
```bash
git push -u origin main
```

## Skills Practiced
- Git Repository Creation
- Tracking Changes
- Creating Commits
- Connecting to GitHub
- Pushing Code to Remote Repository
